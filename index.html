SETUP GUIDE

Supabase 配置说明

1

执行 SQL

在 Supabase SQL Editor 里运行 supabase-setup.sql，创建数据表、存储桶和公开策略。

2

填写配置

打开 config.js，填入你的 supabaseUrl 和 supabaseAnonKey。

3

部署静态站

把整个目录部署到 GitHub Pages、Netlify 或 Vercel，别人就能直接上传和下载。
